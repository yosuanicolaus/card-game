# Card Game

Generate and draw from deck using http://deckofcardsapi.com/ API

## Technology used:

- React
- Bootswatch
- Reactstrap
- Axios
- React Router

## How to play:

Go [here](https://yosuanicolaus.github.io/card-game/) and play blackjack with the computer!
